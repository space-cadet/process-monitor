# T12: Data Export (CSV / Parquet / Grafana)

*Created: 2026-06-18*
*Status: ⬜ PENDING*
*Priority: MEDIUM*

## Problem
Data is locked in SQLite. Users can't analyze it in Python/Pandas, visualize in Grafana, or archive it externally.

## Solution
Multiple export formats and integration points:

### 1. CSV Export (Dashboard)
- Already partially exists: "Export CSV" button for drain events
- Expand to: full snapshot export, process-level export, filtered export

### 2. Parquet Export (CLI)
- `npx tsx src/query.ts --export parquet --since 7d`
- Columnar format, efficient for large datasets
- Ideal for Python/Pandas analysis

### 3. Grafana Integration
- Optional: push metrics to InfluxDB/Prometheus
- Or: expose a simple metrics endpoint (`/metrics` in Prometheus format)

### 4. SQLite Direct Access
- Document the schema so users can query directly
- Provide example queries in README

## Files to Create/Modify
- `src/core/Exporter.ts` — CSV/Parquet generation
- `src/query.ts` — `--export` flag
- `src/web/server.ts` — `GET /api/export?format=csv&since=...`
- `web/public/app.js` — Export buttons
