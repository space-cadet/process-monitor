# T12: Data Export (CSV / JSON)

*Created: 2026-06-18*
*Completed: 2026-06-24*
*Status: ✅ COMPLETE*

## What Was Built

### API Endpoints
- `GET /api/export/csv?from=ISO&to=ISO` — CSV export with date range
- `GET /api/export/json?from=ISO&to=ISO` — JSON export with date range

### CSV Export
- Exports: snapshots, battery data, process samples
- Columns: timestamp, battery%, is_charging, cpu_total, memory_total, process_name, pid, cpu_percent, memory_percent, rss_mb
- Downloadable file with `.csv` extension

### JSON Export
- Structured data: `{snapshots: [...], processes: [...], battery: [...]}`
- Downloadable file with `.json` extension

### Dashboard UI
- **Export section** in Reports tab
- Dropdown: CSV / JSON
- Date range picker: `from` / `to` (ISO 8601)
- Download button triggers file download

## Files
- `src/web/server.ts` — `/api/export/csv`, `/api/export/json` endpoints
- `web/public/app.js` — Export UI rendering + download handling
- `web/public/index.html` — Export dropdown + date inputs
- `web/public/styles.css` — Export section styles
