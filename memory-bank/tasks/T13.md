# T13: Process Tree View (Hierarchical)

*Created: 2026-06-18*
*Completed: 2026-06-24*
*Status: ✅ COMPLETE*

## What Was Built

### API
- `GET /api/process-tree` — returns hierarchical process tree from `systeminformation.processes()`
- Builds tree from `pid` → `ppid` relationships
- Aggregates child CPU/memory into parent totals
- Returns JSON tree: `{name, pid, cpu, mem, rss, children: [...]}`

### Dashboard UI
- **Tree/List toggle** in Processes tab
- **Tree view**: Collapsible, indented hierarchy with aggregate bars
- **List view**: Original flat table (preserved)
- Recursive rendering with CPU/memory per node
- Parent nodes show aggregated totals from children

### Example Tree
```
Visual Studio Code (PID 1234) — 35% CPU, 800 MB
  ├─ Code Helper (Renderer) — 12% CPU, 300 MB
  ├─ Code Helper (GPU) — 8% CPU, 150 MB
  └─ Code Helper (Extension Host) — 15% CPU, 350 MB
```

## Files
- `src/web/server.ts` — `/api/process-tree` endpoint (added to existing server)
- `web/public/app.js` — Tree/List toggle, recursive tree rendering
- `web/public/index.html` — Tree view container
- `web/public/styles.css` — Tree indentation, collapse icons, toggle button

## Dependencies
