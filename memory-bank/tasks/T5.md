## Description
A native macOS Swift app that lives in the menu bar, showing live battery/process status without opening a browser.

## Why Swift?
- Native macOS feel (menu bar icon, native notifications)
- Reads the same SQLite DB — no separate backend needed
- Lower resource usage than a web app
- Can use `IOKit` directly for better energy data

## Features

### Menu Bar Icon
- Battery % with color-coded icon (green/yellow/red)
- Click to open dropdown menu

### Dropdown Menu
- Current battery %, charging status, time remaining
- Top 5 CPU processes right now
- Recent drain events (last 24h)
- "Open Dashboard" → opens web dashboard in browser
- "Settings" → configure thresholds
- "Quit"

### Native Notifications
- Uses `UNUserNotificationCenter` for proper macOS alerts
- Better than osascript: actions ("Open Dashboard", "Dismiss"), grouped notifications

### Design
- Minimal: just the icon + menu, no window unless requested
- Matches macOS design language
