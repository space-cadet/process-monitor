# T1: Enhance mac-process-monitor with battery tracking

*Created: 2026-05-18 18:37 IST*
*Last Updated: 2026-05-18 18:37 IST*

**Status**: 🔄 **IN PROGRESS**
**Priority**: HIGH

## Details

Extend the existing mac-process-monitor Python app to track battery level and detect rapid battery drain (indicating runaway CPU processes). The core concern: when battery starts dropping rapidly, there's likely a process consuming excessive CPU.

## Current State

**Existing components:**
- `procmon/core/collector.py` — Process data collection (CPU, memory, disk via psutil)
- `procmon/core/analyzer.py` — Threshold analysis (CPU > 80%, memory > 15%, duration > 30s)
- `procmon/core/logger.py` — Rotating file logs
- `config/default_config.yaml` — Configuration (thresholds, intervals, ignore lists)

**What's missing:**
- Battery level monitoring
- Battery drain rate calculation
- Alert mechanism (not just file logging)
- Integration with OpenClaw/Telegram for proactive notifications
