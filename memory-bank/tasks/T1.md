## Decisions
- **TypeScript over Python** — Type safety, easier Swift port, better async handling
- **systeminformation library** — Cross-platform, comprehensive, well-maintained
- **better-sqlite3** — Fast synchronous SQLite for time-series data
- **No UI yet** — Console output for now; web dashboard or menubar app later
- **CPU total via si.currentLoad()** — Returns undefined on macOS; using per-process aggregation as fallback

## Known Issues
- `si.currentLoad().currentload` returns undefined on macOS (systeminformation quirk)
- `si.battery().timeRemaining` and `.cycleCount` null on some macOS versions
- Process memoryPercent calculation needs validation against Activity Monitor
