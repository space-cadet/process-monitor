# T10: "What Drained My Battery?" Automated Report

*Created: 2026-06-18*
*Completed: 2026-06-24*
*Status: ✅ COMPLETE*

## What Was Built

### ReportGenerator (`src/core/ReportGenerator.ts`)
- Daily battery health report with scoring algorithm
- Queries: drain events, spikes, battery impact scores from last 24h
- Aggregates per-process impact, finds patterns
- Generates markdown + JSON reports

### CLI
- `npx tsx src/query.ts --report` — generate today's report
- `npx tsx src/query.ts --report --output=json` — JSON format
- `npx tsx src/query.ts --report --date=2026-06-23` — specific date

### Dashboard
- **Reports tab** — health score (0-100), drain events, top culprits, insights
- **Export UI** — dropdown for format (CSV/JSON) + date range picker
- Auto-generates at 22:00 daily via cron

### Report Format
```
📊 Battery Report — June 23, 2026

🔋 Health Score: 85/100
⚠️  Drain Events: 2 (12% total loss)
🔥 Top Spikes: 5 (Chrome x2, Docker x1, VS Code x1, kernel_task x1)
🏆 Top Culprits:
  1. Chrome — 23% battery impact, 4.2h active
  2. Docker Desktop — 18% impact, 2 spikes
  3. VS Code — 8% impact, 1 long spike

💡 Insights:
  • Chrome had 45-min sustained drain at 14:00
  • Docker spiked 3x when on battery
  • No drain during sleep
```
